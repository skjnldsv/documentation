Синхронизация с Windows 10

Календарь

  1. В браузере перейдите в приложение Nextcloud Calendar. В разделе «Настройки и импорт» скопируйте адрес с помощью «Копировать адрес iOS/macOS CalDAV» в буфер обмена.

  2. Запустите приложение «Календарь» в Windows 10. Затем нажмите на значок настроек (значок шестерёнки) и выберите «Управление учётными записями».

  3. Нажмите «Добавить аккаунт» и выберите «iCloud».

  4. Введите электронную почту, имя пользователя и пароль. Все эти данные временные и они будут изменены на последующих этапах.

  5. Нажмите «Готово». Должно появиться сообщение о том, что настройки были успешно сохранены.

  6. В меню «Управление учетными записями» щелкните учетную запись iCloud, созданную на предыдущих этапах, и выберите «Изменить настройки». Затем нажмите «Изменить настройки синхронизации почтового ящика».

  7. Прокрутите диалоговое окно до самого низа, выберите «Дополнительные настройки почтового ящика». Ещё раз прокрутите диалоговое окно до самого низа и вставьте URL-адрес CalDAV в поле «Сервер календаря (CalDAV)».

  8. Нажмите «Готово». Введите имя пользователя и пароль Nextcloud в соответствующие поля и измените имя учётной записи на любое удобное для вас (например, «Nextcloud Calendar»). Нажмите «Сохранить».

Контакты

  1. В нижней левой части окна «Контакты» (в Nextcloud Contacts) найдите маленький символ цепи, который выглядит следующим образом:

который отобразит URL, выглядящий примерно так: https://cloud.nextcloud.com/remote.php/dav/addressbooks/users/daniel/Thunderbird/

  1. Запустите приложение «Календарь» в Windows 10. Затем нажмите на значок настроек (значок шестерёнки) и выберите «Управление учётными записями».

  2. Нажмите «Добавить аккаунт» и выберите «iCloud».

  3. Введите электронную почту, имя пользователя и пароль. Все эти данные временные и они будут изменены на последующих этапах.

  4. Нажмите «Войти», а затем «Готово». Должно появиться сообщение об успешном сохранении настроек.

  5. В меню «Управление учетными записями» щелкните учетную запись iCloud, созданную на предыдущих этапах, и выберите «Изменить настройки». Затем нажмите «Изменить настройки синхронизации почтового ящика».

  6. Прокрутите диалоговое окно до самого низа, выберите «Дополнительные настройки почтового ящика». Ещё раз прокрутите диалоговое окно до самого низа и вставьте URL-адрес CardDAV в поле «Сервер контактов (CardDAV)».

  7. Нажмите «Готово». Введите имя пользователя и пароль Nextcloud в соответствующие поля и измените имя учётной записи на любое удобное для вас (например, «Nextcloud Contacts»). Нажмите «Сохранить».

Возможные проблемы

После выполнения всех этих шагов ваш календарь Nextcloud должен синхронизироваться. Если нет, проверьте свое имя пользователя и пароль. В противном случае повторите эти шаги.

ПРИМЕЧАНИЕ. Вы не сможете синхронизировать свой календарь, если у вас включена двухфакторная аутентификация. Выполните следующие действия, чтобы получить пароль приложения, который можно использовать с клиентским приложением Календарь:

  1. Войдите в Nextcloud. Нажмите на свой значок пользователя, затем нажмите «Настройки».

  2. Нажмите «Безопасность», затем найдите кнопку с надписью «Создать пароль приложения». Рядом с этой кнопкой введите «Приложение Календарь Windows 10». Затем нажмите кнопку и скопируйте и вставьте пароль. Используйте этот пароль вместо вашего пароля Nextcloud для шага 8.

Особая благодарность этому пользователю Reddit за его пост <https://www.reddit.com/r/Nextcloud/comments/5rcypb/using_the_windows_10_calendar_application_with/>`_

Контакты

  1. Повторите шаги 1-7 из инструкции по работе с Календарём. Если вы уже настроили синхронизацию Календаря, вы можете использовать для этого ту же учётную запись.

  2. В разделе «Дополнительные настройки почтового ящика» вставьте URL-адрес CalDAV в поле с надписью «Сервер контактов (CardDAV)».

  3. Замените путь «principals» в URL на «addressbooks».

  4. Нажмите «Готово». Введите имя пользователя и пароль Nextcloud в соответствующие поля и измените имя учётной записи на любое удобное для вас (например, «Nextcloud»). Нажмите «Сохранить».
