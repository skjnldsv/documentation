普遍存取

普遍存取對我們來說非常重要。我們遵循網頁標準，並檢查所有內容是否可用於鍵盤和輔助軟件，例如螢幕閱讀器。我們的目標是符合 網頁內容無障礙指引 2.1 的 AA 級別，並且在 AAA 級別上也有高對比度主題。我們還遵循德國的 BITV 2.0 指引。

如果您發現任何問題，請隨時在 我們的問題追蹤器 上報告。如果您想參與，歡迎 加入我們的設計團隊！

縮放與反應性

Nextcloud 介面完全響應，適用於任何大小的螢幕。您可以進行縮放，以符合您的文本和元素大小。導航和側邊欄可以展開或折疊。

透過鍵盤導航

您可以僅使用鍵盤導航網頁介面，就像使用滑鼠一樣：

  • Tab 和 Shift + Tab 在元素之間移動

  • Enter 或 Space 以啟動或打開元素（根據元素的類型而定）

  • Escape 用於關閉模態窗口、彈出選項單和檔案查看器

  • 左箭頭 和 右箭頭 在查看器中導航照片

  • Ctrl + F 專注於搜索欄

  • Ctrl + S 在編輯器（如 Nextcloud Text）中保存更改

為了更快速的導航，我們在文檔開頭提供了 2 個「跳過連結」，讓您可以：

  • 跳到主要內容

  • 跳到應用的導航

Nextcloud Talk 有快捷鍵，這些快捷鍵也在應用的設置中有記錄：

  • C 專注於消息輸入框

  • Escape 取消專注於消息輸入框，以便使用快捷鍵

  • F 全螢幕聊天或通話

  • 通話中：

    • M 切換麥克風開關

    • V 切換視頻開關

    • Space 用於按鍵通話或靜音

    • R 舉手或放下手

Nextcloud Mail 也有快捷鍵，這些快捷鍵在應用的設置中有記錄：

  • C 撰寫新消息

  • 左箭頭 切換到較新的消息

  • 右箭頭 切換到較舊的消息

  • S 將消息標記為收藏

  • U 將消息標記為未讀

  • Del 刪除消息

  • Ctrl + Enter 發送

  • R 刷新並加載新郵件

包含的主題

我們提供幾個可以啟動的主題，以幫助提升可及性：

  • 高對比度主題： 一種高對比度模式，方便您的導航。視覺質量將降低，但清晰度將提高。

  • 黑暗主題： 一種黑暗主題，通過降低整體亮度來保護您的眼睛。此主題仍在開發中，請報告任何您可能發現的問題。

  • 閱讀障礙字體： OpenDyslexic 是一種免費字體，旨在減少由閱讀障礙引起的一些常見閱讀錯誤。

要進入無障礙設置：

  1. 打開標題末尾的設置選項單

  2. 選擇 設置

  3. 在導航中選擇 無障礙性

Note

元素的對比度可能會根據自定義主題而有所不同。例如，主題顏色用作標題、登錄頁面和主要按鈕的背景顏色。如果這造成對比問題，請聯繫您的管理員以獲取幫助。
