Fil Deling

Nextcloud brukarar kan dele filer og mapper. Moglege mål er:

  • offentlege linker

  • brukarar

  • grupper

  • sirkler

  • snakke samtaler

  • brukarar eller grupper på føderale Nextcloud tenarar

Note

Some options may not be available due to administrative configuration. See administrator documentation for details.

Offentlege link delingar

Du kan dele filer og mapper gjennom offentlege lenker.

Ein tilfeldig 15-siffret token vil bli skapt. Lenken vil sjå ut som https://cloud.example.com/s/yxcFKRWBJqYYzp4.

Eit rekke valg er tilgjengeleg for offentlege mappe delingar:

  • Berre les for å tillate visning og nedlasting

  • Tillat opplasting og redigering

  • Med Fil slipp, kan delaren berre laste opp filer til ei mappe utan å sjå filene som er i mappa frå før av.

  • Hide download hides the download buttons and the default browser right-click options in order to make downloading for the sharee harder

  • Passordbeskytte

  • Sett utgongsdato vil automatisk deaktivere delingen

  • Merknad til mottakaren

  • Slutt å dele for å tilbakestille delingen

  • Legg til ei annan lenke for å skape fleire offentlege lenker med ulike rettigheiter

For offentleg fil delinger, kan du tillate redigering av fila med ein av Nextcloud sine samarbeidende redigerings løysingar:

Note

Password protection as well as file expiration are also propagated using Federated File Sharing since Nextcloud 22.

Interne delinger med brukarar og grupper

Når du deler med brukarar, grupper, sirkler eller medlemmer av ein Snakke samtale, er rettigheta til filene eller mappen sitt innhald regulerbart:

Som ein delar, kan du konfigurere om du automatisk vil akseptere alle innkommende delinger og ha dei lagt til i di rot mappe, eller om du vil bli spurt kvar gong om du vil akseptere eller avvise delingen.

For å regulere akseptering innstillingen, gå til Innstillingar > Personleg > Deling:

Warning

If the owner renames a shared file or folder, the new name will not be reflected on the recipient’s side. This is needed to prevent overwriting existing files or folders on the recipient’s end.

Andre med tilgong

For å kunne finne ut om ei fil eller mappe er tilgjengeleg for andre gjennom deling av ein overordnet mappe hierarki nivå, trykk på Andre med aksess i delingsfanen:

Lista viser alle brukarar, grupper, chatter etc. som det noverande objektet har fått tilgong til gjennom deling av ein overlegen mappe i hierarkiet:

Trykk på dei tre dottane for å:

  • sjå kven som satt i gong delingen

  • sjå kvar delingen vart satt i gong (trykk for å navigere til mappen, så langt du har tilgong der)

  • Slutt å dele den fyrste delinga (berre tilgjengeleg for dele eigaren)

Note

Denne informasjonen er berre synleg for eigaren av ei fil/mappe eller delarar med gjendeling rettigheiter.

Fødererte Delingar

Føderal Deling tillater deg å montere fil delinger frå eksterne Nextcloud tenarar, der du då skaper di eigen sky av Nextcloud. Du kan skape direkte dele lenker med brukarar på andre Nextcloud tenarar.

Oppretting av ein ny Føderal Deling

Føderal deling er aktivert som standard. Følg desse stega for å skape ein ny deling med andre Nextcloud eller ownCloud tenarar:

Gå til dine «Filer» siden og trykk på Dele ikonet på filen eller katalogen som du vil dele. I sidefeltet legger du inn brukarnamnet og URL en til den eksterne brukeren i denne formen: «<username>@<nc-server-url>». I dette eksemplet, som er bob@cloud.example.com:
