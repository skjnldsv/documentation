Synchroniser avec iOS

Agenda

  1. Ouvrir l’application paramètres.

  2. Select Calendar.

  3. Select Accounts.

  4. Sélectionner Ajouter un Compte.

  5. Choisir Autre comme type de compte.

  6. Choisir Ajouter un compte CalDAV.

  7. Pour le serveur, tapez le nom de domaine de votre serveur, c’est-à-dire « exemple.com ».

  8. Entrez votre nom d’utilisateur et votre mot de passe.

  9. Choisir Suivant.

Votre calendrier sera désormais visible dans l’application Calendrier.

Note

À partir de l’iOS 12, un cryptage SSL est nécessaire. Par conséquent, ne désactivez pas le SSL (pour cette raison, un certificat est nécessaire pour votre domaine, https://letsencrypt.org/ fera l’affaire).

Contacts

  1. Ouvrir l’application paramètres.

  2. Select Contacts.

  3. Select Accounts.

  4. Sélectionner Ajouter un Compte.

  5. Choisir Autre comme type de compte.

  6. Sélectionner Ajouter un compte CardDAV.

  7. Pour le serveur, tapez le nom de domaine de votre serveur, c’est-à-dire « exemple.com ».

  8. Entrez votre nom d’utilisateur et votre mot de passe.

  9. Choisir Suivant.

Vous devriez maintenant trouver vos contacts dans le carnet d’adresses de votre iPhone.

Note

À partir de l’iOS 12, un cryptage SSL est nécessaire. Par conséquent, ne désactivez pas le SSL (pour cette raison, un certificat est nécessaire pour votre domaine, https://letsencrypt.org/ fera l’affaire).

Si cela ne fonctionne toujours pas, consultez le guide « Contacts et calendrier de dépannage ».
