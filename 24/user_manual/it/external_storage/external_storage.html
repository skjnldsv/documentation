Configurare l’archiviazione esterna

L’applicazione External Storage vi permette di montare servizi di storage esterni, come Amazon S3, SMB/CIFS fileservers e server FTP …in Nextcloud. Il vostro amministratore del server Nextcloud gestisce quali di questi servizi sono disponibili per voi. Vi prego di vedere Configuring External Storage (GUI)  nel manuale per amministratore di Nextcloud per guide di configurazioni ed esempi.
