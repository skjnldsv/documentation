Sincronizando con iOS

Calendario

  1. Abra a aplicación de axustes.

  2. Select Calendar.

  3. Select Accounts.

  4. Seleccione Engadir conta.

  5. Seleccione Outro como tipo de conta.

  6. Seleccione Engadir conta CalDAV.

  7. Para o servidor, escriba o nome de dominio do servidor, é dicir exemplo.com.

  8. Introduza o seu nome de usuario e contrasinal.

  9. Seleccione Seguinte.

O seu calendario agora será visíbel na aplicación Calendario.

Note

A partir do iOS 12 é necesario un cifrado SSL. Polo tanto, non desactive SSL (Por este motivo é necesario que o seu dominio dispoña dun certificado, pode obtelo en: https://letsencrypt.org/).

Contactos

  1. Abra a aplicación de axustes.

  2. Select Contacts.

  3. Select Accounts.

  4. Seleccione Engadir conta.

  5. Seleccione Outro como tipo de conta.

  6. Seleccione Engadir conta CardDAV.

  7. Para o servidor, escriba o nome de dominio do servidor, é dicir exemplo.com.

  8. Introduza o seu nome de usuario e contrasinal.

  9. Seleccione Seguinte.

Agora debe atopar os seus contactos na axenda de enderezos do seu iPhone.

Note

A partir do iOS 12 é necesario un cifrado SSL. Polo tanto, non desactive SSL (Por este motivo é necesario que o seu dominio dispoña dun certificado, pode obtelo en: https://letsencrypt.org/).

Se aínda non funciona, bótelle un ollo á guía Resolución de problemas de Contactos e Calendario.
