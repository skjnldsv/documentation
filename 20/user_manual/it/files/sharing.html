Condivisione file

Gli utenti Nextcloud possono condividere file e cartelle. Possibili destinatari sono

  • collegamenti pubblici

  • utenti

  • gruppi

  • cerchie

  • conversazioni talk

  • utenti e gruppi nei server Nextcloud federati

Nota

Alcune opzioni potrebbero non essere disponibili a causa della configurazione amministrativa. Vedi admin documentation per i dettagli.

Condivisione di collegamenti pubblici

Puoi condividere file e cartelle tramite collegamenti pubblici.

Un token di 15 caratteri random verrà creato. Il collegamento sarà simile a https://cloud.example.com/s/yxcFKRWBJqYYzp4.

Diverse opzioni saranno disponibili per le condivisioni di cartelle pubbliche:

  • Sola lettura per consentire la visualizzazione e il download

  • Consenti caricamento e modifica

  • Con Rilascia file, il destinatario può solo caricare file in una cartella senza vedere i file già presenti nella cartella.

  • Nascondi download impedisce al destinatario di scaricare

  • Proteggi con password

  • Imposta data di scadenza disattiverà automaticamente la condivisione

  • Nota per il destinatario

  • Annulla la condivisione per revocare una condivisione

  • Aggiungi un altro collegamento per creare multipli collegamenti pubblici con privilegi differenti

Per condivisioni di file pubblici, puoi consentire la modifica del file con una delle soluzioni di editing collaborative di Nextcloud.

Nota

Password protection as well as file expiration are not propagated using Federated File Sharing in current Nextcloud releases. This will be adjusted in Nextcloud 22.

Condivisioni interne con utenti e gruppi

Quando condividi con utenti, gruppi, circoli o membri di una conversazione Talk, i privilegi per i file e i contenuti delle cartelle sono regolabili:

Come destinatario della condivisione, puoi configurare se vuoi automaticamente accettare tutte le condivisioni in entrata e averle aggiunte alla tua cartella di root, o se vuoi essere domandato ogni volta se vuoi accettare o declinare la condivisione.

Per regolare l’impostazione sull’accettazione, vai a Impostazioni > Personale > Condivisione

Altri con accesso

Per scoprire se un file o una cartella è accessibile ad altri tramite la condivisione di una cartella di livello gerarchico superiore, fai clic su Altri con accesso nella scheda di condivisione.

La lista mostra tutti gli utenti, i gruppi, le chat etc. a cui è stato consentito l’accesso all’oggetto corrente tramite la condivisione di una cartella di livello gerarchico superiore.

Fai clic sui tre puntini per

  • vedere chi ha iniziato la condivisione

  • vedi dove è iniziata la condivisione (fai clic per navigare fino alla cartella, se puoi accedervi)

  • Annulla la condivisione iniziale (accessibile solo per il proprietario della condivisione)

Nota

Questa informazione è visibile solo per il proprietario di un file/una cartella o destinatari con privilegi di ricondivisione.

Condivisioni federate

La Condivisione federata ti consente di montare una cartella condivisa di file da server Nextcloud remoti, effettivamente creando il tuo proprio cloud di Nextcloud. Puoi creare collegamenti di condivisione diretti con gli utenti sugli altri server Nextcloud.

Creare una nuova Condivisione federata

La Condivisione federata è abilitata in modo predefinito. Segui questi passaggi per creare una nuova condivisione con altri server Nextcloud o ownCloud:

Vai alla tua pagina File e fai clic sull’icona di condivisione sul file o sulla directory che vuoi condividere. Nella barra laterale inserisci il nome utente e l’URL dell’utente remoto in questo modulo: <username>@<nc-server-url>. In questo esempio, esso è bob@cloud.esempio.com.

Il destinatario riceve una notifica nel proprio Nextcloud, consentendogli di o accettare o declinare la condivisione in entrata.

Aggiungere una risorsa pubblica al proprio Nextcloud
