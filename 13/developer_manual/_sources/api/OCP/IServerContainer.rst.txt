IServerContainer
================


.. php:namespace:: OCP

.. php:interface:: IServerContainer


	.. rst-class:: phpdoc-description
	
		| This is a tagging interface for the server container
		
		| The interface currently extends IContainer, but this interface is deprecated as of Nextcloud 20,
		| thus this interface won\'t extend it anymore once that was removed\. So migrate to the ContainerInterface
		| only\.
		
	
	:Source:
		`lib/public/IServerContainer.php#61 <https://github.com/nextcloud/server/blob/stable13/lib/public/IServerContainer.php#L61>`_
	
	:Parent:
		:php:interface:`Psr\\Container\\ContainerInterface`
	
	:Parent:
		:php:interface:`OCP\\IContainer`
