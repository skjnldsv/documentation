Svg color api

More informations about scss and this api: scss mixins and functions

You can request and color any svg icons used in nextcloud with this api. The server will directly change the colours of the circle, rect and path elements in the svg you provide. Simply use those urls:

  • https://yourdomain/svg/core/actions/menu?color=ffffff Will serve the svg located in the core/img directory as a white icon /core/img/actions/menu.svg

  • https://yourdomain/svg/core/places/calendar?color=0082c9 Will serve the svg located in the core/img directory with the color #0082c9 /core/img/places/calendar.svg

  • https://yourdomain/svg/files/app?color=000000 Will serve the svg located in the files app img directory ad a black icon /app/files/img/app.svg
