Acceso universal

O acceso universal é moi importante para nós. Seguimos os estándares web e comprobamos que todo sexa utilizábel tamén con teclado e software de asistencia como os lectores de pantalla. Pretendemos cumprir as Directrices de accesibilidade ao contido web 2.1 no nivel AA, co tema de alto contraste incluso no nivel AAA. Tamén seguimos as directrices alemás BITV 2.0.

Se atopa algún problema, non dubide en comunicarllo no noso rastrexador de problemas. E se quere participar, únase ao noso equipo de deseño!

Zoom e capacidade de resposta

A interface de Nextcloud é totalmente sensíbel e pode usarse en calquera tamaño de pantalla. Pode ampliar e reducir o tamaño do texto e dos elementos ao seu gusto. A navegación e a barra lateral pódense expandirse ou contraerse.

Navegando a través do teclado

Pode navegar pola interface web só co teclado igual que o fai co rato:

  • Tab e Maiús + Tab moverse entre os elementos

  • Intro ou Espazo para activar ou abrir o elemento (dependendo do tipo de elemento)

  • Escape empregase para pechar modais, menús emerxentes e visores de ficheiros

  • Frecha esquerda Frecha dereita navegar entre as fotos no visor

  • Ctrl + F para centrar o campo de busca

  • Ctrl + S para gardar cambios nos editores como Nextcloud Text

Para unha navegación máis rápida, ofrecemos 2 «ligazóns de salto» ao comezo do documento que lle permiten:

  • Ir ao contido principal

  • Saltar á navegación da aplicación

Nextcloud Talk ten accesos directos que tamén están documentados na configuración da propia aplicación:

  • C para poñer en foco o campo de entrada da mensaxe

  • Escape para quitar de foco o campo de entrada de mensaxes para poder usar atallos

  • F para pantalla completa da conversa ou chamada

  • Durante unha chamada:

    • M para activar e desactivar o micrófono

    • V para activar e desactivar o vídeo

    • Espazo para premer para falar ou premer para silenciar

    • R para erguer ou baixar a man

Nextcloud Mail tamén ten atallos, documentados que tamén están na configuración da propia aplicación:

  • C para redactar unha nova mensaxe

  • Frecha esquerda cambiar a unha mensaxe máis recente

  • Frecha dereita cambiar a unha mensaxe máis antiga

  • S para marcar unha mensaxe como favorita

  • U para marcar unha mensaxe coma non lido

  • Supr para eliminar unha mensaxe

  • Ctrl + Intro para enviar

  • R para actualizar e cargar novos correos

Temas incluídos

Ofrecense varios temas que pode activar para facilitar a accesibilidade:

  • Tema de alto contraste: Un modo de alto contraste para facilitar a navegación. A calidade visual reducirase mais aumentará a claridade.

  • Tema escuro: Un tema escuro para aliviar os seus ollos reducindo a luminosidade e o brillo xeral. Aínda está en desenvolvemento, así que informe de calquera problema que poida atopar.

  • Tipo de letra dislexia: OpenDyslexic é un tipo de letra libre e debalde deseñado para mitigar algúns dos erros de lectura máis comúns causados ​​pola dislexia.

Para acceder á configuración de accesibilidade:

  1. Abra o menú de axustes ao final da cabeceira

  2. Escolla Axustes

  3. Na navegación, escolla Accesibilidade

Note

O contraste dos elementos pode variar en función da temática personalizada. Por exemplo, a cor temática principal úsase como cor de fondo na cabeceira, a páxina de inicio de sesión e os botóns principais. Se isto causa problemas de contraste, póñase en contacto co seu administrador para obter axuda.
