EmptyContentSecurityPolicy
==========================


.. php:namespace:: OCP\AppFramework\Http

.. php:class:: EmptyContentSecurityPolicy


	.. rst-class:: phpdoc-description
	
		| Class EmptyContentSecurityPolicy is a simple helper which allows applications
		| to modify the Content\-Security\-Policy sent by ownCloud\. Per default the policy
		| is forbidding everything\.
		
		| As alternative with sane exemptions look at ContentSecurityPolicy
		
	
	:Source:
		`lib/public/AppFramework/Http/EmptyContentSecurityPolicy.php#40 <https://github.com/nextcloud/server/blob/stable13/lib/public/AppFramework/Http/EmptyContentSecurityPolicy.php#L40>`_
