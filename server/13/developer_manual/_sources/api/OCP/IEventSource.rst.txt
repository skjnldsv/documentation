IEventSource
============


.. php:namespace:: OCP

.. php:interface:: IEventSource


	.. rst-class:: phpdoc-description
	
		| wrapper for server side events \(http://en\.wikipedia\.org/wiki/Server\-sent\_events\)
		| includes a fallback for older browsers and IE
		
		| use server side events with caution, to many open requests can hang the server
		| 
		| The event source will initialize the connection to the client when the first data is sent
		
	
	:Implemented by:
		:php:class:`OC\_EventSource` 
	
	
	:Source:
		`lib/public/IEventSource.php#35 <https://github.com/nextcloud/server/blob/stable13/lib/public/IEventSource.php#L35>`_
