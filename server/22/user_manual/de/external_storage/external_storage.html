Externen Speicher konfigurieren

Mit der Anwendung External Storage können Sie externe Speicherdienste wie Amazon S3, SMB / CIFS-Dateiserver und FTP-Server in Nextcloud bereitstellen. Ihr Nextcloud-Serveradministrator steuert, welche Ihnen davon zur Verfügung stehen. Informationen zur Konfiguration mit Beispielen und HowTos finden Sie unter Konfigurieren des externen Speichers (GUI) <https://docs.nextcloud.com/server/latest/admin_manual/configuration_files/external_storage_configuration_gui.html> _ im Nextcloud-Administratorenhandbuch.
