Compartir ficheiros

Os usuarios do Nextcloud poden compartir ficheiros e cartafoles. Os obxectivos posíbeis son:

  • ligazóns públicas

  • usuarios

  • grupos

  • círculos

  • conversas no Talk

  • usuarios ou grupos en servidores do Nextcloud federados

Note

É probábel que algunhas opcións estean dispoñíbeis, debido á configuración administrativa. Vexa a documentación do administrador para máis detalles.

Compartir con ligazóns públicas

Podes compartir ficheiros e cartafoles a través de ligazóns públicas.

Crearase un testemuño de 15 díxitos ao chou. A ligazón semallará https://cloud.exemplo.com/s/yxcFKRWBJqYYzp4`.

Hai varias opcións dispoñíbeis para as comparticións de cartafol público:

  • Só lectura para permitir a súa vista e descarga

  • Permitir o envío e a edición

  • Con Soltar ficheiro, o receptor só pode enviar ficheiros a un cartafol sen ver os ficheiros que xa están nese cartafol.

  • Hide download hides the download buttons and the default browser right-click options in order to make downloading for the sharee harder

  • **Protexido con contrasinal*

  • Estabelecer a data de caducidade desactivará automaticamente a compartición

  • Nota para o destinatario

  • Deixar de compartir para reverter a compartición

  • Engadir outra ligazón para crear varias ligazóns públicas con diferentes dereitos

Note

Password protection as well as file expiration are not propagated using Federated File Sharing in current Nextcloud releases. This will be adjusted in Nextcloud 22.

Outros con acceso

Para saber se un ficheiro ou cartafol é accesíbel a outros mediante a compartición dun nivel superior de xerarquía de cartafoles, prema en Outros con acceso na lapela de compartidos:

A lista amosa a todos os usuarios, grupos, conversas, etc., aos que se lles permitiu acceder ao obxecto actual mediante a compartición dun cartafol superior na xerarquía:

Prema nos tres puntos para:

  • ver quen iniciou a compartición

  • ver onde se iniciou a compartición (prema para ir ata o cartafol, ata onde teña acceso)

  • Deixar de compartir o recurso inicial (accesíbel só para o propietario do recurso)

Note

Esta información só é visible para o propietario dun ficheiro/cartafol ou receptores con permisos para volver compartir.

Comparticións federadas

A compartición federada permítelle montar ficheiros compartidos dende servidores Nextcloud remotos, creando así a súa propia nube de Nextclouds. Pode ligazóns directas de compartición cos usuarios noutros servidores Nextcloud.
