Κοινή χρήση αρχείων

Nextcloud users can share files and folders. Possible targets are:

  • δημόσιοι σύνδεσμοι

  • χρήστες

  • ομάδες

  • κύκλους

  • συζήτηση συνομιλιών

  • χρήστες ή ομάδες σε ενοποιημένους διακομιστές Nextcloud

Σημείωση

Some options may not be available due to administrative configuration. See admin documentation for details.

Δημόσιες κοινοποιήσεις συνδέσμων

Μπορείτε να μοιραστείτε αρχεία και φακέλους μέσω δημόσιων συνδέσμων.

Θα δημιουργηθεί ένα τυχαίο 15ψήφιο διακριτικό. Ο σύνδεσμος θα μοιάζει με «https: // cloud.example.com / s / yxcFKRWBJqYYzp4».

Διατίθενται πολλές επιλογές για κοινόχρηστο κοινόχρηστο * φάκελο *:

  • ** Μόνο για ανάγνωση ** για να επιτρέπεται η προβολή και η λήψη

  • Allow upload and editing

  • With File drop, the sharee can only upload files to a folder without seeing the files that are already in that folder.

  • Hide download hides the download buttons and the default browser right-click options in order to make downloading for the sharee harder

  • ** Προστασία με κωδικό πρόσβασης **

  • ** Ορισμός ημερομηνίας λήξης ** θα απενεργοποιήσει αυτόματα την κοινή χρήση

  • ** Σημείωση στον παραλήπτη **

  • ** Καταργήστε την κοινή χρήση ** για να επαναφέρετε την κοινή χρήση

  • ** Προσθέστε έναν άλλο σύνδεσμο ** για να δημιουργήσετε πολλούς δημόσιους συνδέσμους με διαφορετικά δικαιώματα

For public file shares, you may allow editing the file with one of Nextcloud’s collaborative editing solutions:

Σημείωση

Password protection as well as file expiration are not propagated using Federated File Sharing in current Nextcloud releases. This will be adjusted in Nextcloud 22.

Εσωτερικές κοινοποιήσεις με χρήστες και ομάδες

Κατά την κοινή χρήση με χρήστες, ομάδες, κύκλους ή μέλη μιας συνομιλίας στο Talk, τα δικαιώματα για τα αρχεία ή το περιεχόμενο του φακέλου είναι ρυθμιζόμενα:

Ως κοινόχρηστο στοιχείο, μπορείτε να διαμορφώσετε εάν θέλετε αυτόματα να αποδεχτείτε όλες τις εισερχόμενες κοινοποιήσεις και να τις προσθέσετε στον ριζικό σας φάκελο ή εάν θέλετε να ερωτηθείτε κάθε φορά εάν θέλετε να αποδεχτείτε ή να απορρίψετε την κοινή χρήση.

For adjusting the acceptance setting, go to Settings > Personal > Sharing:

Άλλοι με πρόσβαση

In order to find out if a file or folder is accessible to others through sharing of a superior folder hierarchy level, click on Others with access in the sharing tab:

The list shows all users, groups, chats etc. that the current object has been given access to through sharing of a superior folder in the hierarchy:

Click on the three dots to:

  • δείτε ποιος ξεκίνησε την κοινή χρήση

  • δείτε πού ξεκίνησε η κοινή χρήση (κάντε κλικ για πλοήγηση στο φάκελο, εφόσον έχετε πρόσβαση εκεί)

  • καταργήστε την κοινή χρήση του αρχικού μεριδίου (προσβάσιμο μόνο για τον κάτοχο της κοινής χρήσης)

Σημείωση

Αυτές οι πληροφορίες είναι ορατές μόνο στον κάτοχο ενός αρχείου / φακέλου ή μοιράζονται με δικαιώματα κοινής χρήσης.

Ομόσπονδες μετοχές

Η Ομοσπονδία κοινής χρήσης σάς επιτρέπει να τοποθετείτε κοινόχρηστα αρχεία από απομακρυσμένους διακομιστές Nextcloud, δημιουργώντας ουσιαστικά το δικό σας σύννεφο Nextclouds. Μπορείτε να δημιουργήσετε απευθείας συνδέσμους κοινής χρήσης με χρήστες σε άλλους διακομιστές Nextcloud.

Δημιουργία νέας κοινής χρήσης

Η κοινή χρήση ομοσπονδίας είναι ενεργοποιημένη από προεπιλογή. Ακολουθήστε αυτά τα βήματα για να δημιουργήσετε μια νέα κοινή χρήση με άλλους διακομιστές Nextcloud ή ownCloud:

Go to your Files page and click the Share icon on the file or directory you want to share. In the sidebar enter the username and URL of the remote user in this form: <username>@<nc-server-url>. In this example, that is bob@cloud.example.com:

The sharee is receiving a notification in their Nextcloud, allowing them to either accept or decline the incoming share:

Προσθήκη δημόσιας κοινής χρήσης στο Nextcloud σας

Οι σελίδες κοινής χρήσης δημόσιου συνδέσμου Nextcloud προσφέρουν την επιλογή προσθήκης αυτού του αρχείου ή φακέλου ως ενοποιημένης κοινής χρήσης στη δική σας παρουσία Nextcloud. Απλώς εισαγάγετε το «<όνομα χρήστη> @ <nc-server-url>» όπως ακριβώς εμφανίζεται για τις εξερχόμενες μετοχές παραπάνω:
