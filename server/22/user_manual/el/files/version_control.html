Έλεγχος έκδοσης

Nextcloud supports simple version control system for files. Versioning creates backups of files which are accessible via the Versions tab on the Details sidebar. This tab contains the history of the file where you can roll back a file to any previous version. Changes made at intervals greater than two minutes are saved in data/[user]/files_versions.

Για να επαναφέρετε μια συγκεκριμένη έκδοση ενός αρχείου, κάντε κλικ στο κυκλικό βέλος προς τα αριστερά. Κάντε κλικ στη χρονική σήμανση για να το κατεβάσετε.

Η εφαρμογή εκδόσεων λήγει αυτόματα παλιές εκδόσεις για να βεβαιωθείτε ότι ο χρήστης δεν έχει εξαντληθεί ο χώρος. Αυτό το μοτίβο χρησιμοποιείται για τη διαγραφή παλαιών εκδόσεων:

  • Για το πρώτο δευτερόλεπτο διατηρούμε μια έκδοση

  • Για τα πρώτα 10 δευτερόλεπτα το Nextcloud διατηρεί μία έκδοση κάθε 2 δευτερόλεπτα

  • Για το πρώτο λεπτό, το Nextcloud διατηρεί μία έκδοση κάθε 10 δευτερόλεπτα

  • Για την πρώτη ώρα, το Nextcloud διατηρεί μία έκδοση κάθε λεπτό

  • Για τις πρώτες 24 ώρες το Nextcloud διατηρεί μία έκδοση κάθε ώρα

  • Για τις πρώτες 30 ημέρες το Nextcloud διατηρεί μία έκδοση κάθε μέρα

  • Μετά τις πρώτες 30 ημέρες το Nextcloud διατηρεί μία έκδοση κάθε εβδομάδα

Οι εκδόσεις προσαρμόζονται σε αυτό το μοτίβο κάθε φορά που δημιουργείται μια νέα έκδοση.

Η εφαρμογή έκδοσης δεν χρησιμοποιεί ποτέ περισσότερο από το 50% του διαθέσιμου ελεύθερου χώρου του χρήστη. Εάν οι αποθηκευμένες εκδόσεις υπερβαίνουν αυτό το όριο, το Nextcloud διαγράφει τις παλαιότερες εκδόσεις έως ότου εκπληρώσει ξανά το όριο χώρου στο δίσκο.
