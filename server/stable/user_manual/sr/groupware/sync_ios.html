Синхронизација са iOS

Calendar

  1. Отворите апликацију подешавања.

  2. Select Apps.

  3. Изаберите Calendar.

  4. Select Calendar Accounts.

  5. Изаберите Add Account.

  6. За тип налога изаберите Other.

  7. Изаберите Add CalDAV account.

  8. За сервер, откуцајте име домена вашег сервера, нпр. example.com.

  9. Унесите своје корисничко име и лозинку.

  10. Изаберите Next.

  11. Open Advanced Settings

  12. For server, type the domain name of your server and username i.e. example.com/remote.php/dav/principals/users/username/

  13. Close Advanced Settings

Сада ће ваш календар бити видљив у апликацији Calendar.

Белешка

Почевши са iOS 12 неопходно је SSL шифровање. Стога немојте да искључите SSL (Из овог разлога је потребно да ваш домен има сертификат, https://letsencrypt.org/ ће бити довољно).

Contacts

  1. Отворите апликацију подешавања.

  2. Select Apps.

  3. Изаберите Contacts.

  4. Select Contacts Accounts.

  5. Изаберите Add Account.

  6. За тип налога изаберите Other.

  7. Изаберите Add CardDAV account.

  8. За сервер, откуцајте име домена вашег сервера, нпр. example.com.

  9. Унесите своје корисничко име и лозинку.

  10. Изаберите Next.

Сада би требало да пронађете контакте у адресару вашег iPhone телефона.

Белешка

Почевши са iOS 12 неопходно је SSL шифровање. Стога немојте да искључите SSL (Из овог разлога је потребно да ваш домен има сертификат, https://letsencrypt.org/ ће бити довољно).

Ако још увек не функционише, Погледајте Решавање проблема са Contacts & Calendar или Решавање проблема са откривањем сервиса.
