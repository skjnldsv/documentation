Sincronizando com iOS

Calendário

  1. Abra o aplicativo de configurações.

  2. Select Apps.

  3. Selecione Calendário.

  4. Select Calendar Accounts.

  5. Selecione Adicionar conta.

  6. Selecione Outro como tipo de conta.

  7. Selecione Adicionar conta CalDAV.

  8. Para servidor, digite o nome de domínio do seu servidor, ou seja, example.com.

  9. Digite seu nome de usuário e senha.

  10. Selecione Avançar.

  11. Open Advanced Settings

  12. For server, type the domain name of your server and username i.e. example.com/remote.php/dav/principals/users/username/

  13. Close Advanced Settings

Sua agenda agora estará visível no aplicativo Agenda.

Nota

A partir do iOS 12, é necessária uma criptografia SSL. Portanto, não desative o SSL (por esse motivo, um certificado é necessário em seu domínio, https://letsencrypt.org/ o fará).

Contatos

  1. Abra o aplicativo de configurações.

  2. Select Apps.

  3. Selecione Contatos.

  4. Select Contacts Accounts.

  5. Selecione Adicionar conta.

  6. Selecione Outro como tipo de conta.

  7. Selecione Adicionar conta CardDAV.

  8. Para servidor, digite o nome de domínio do seu servidor, ou seja, example.com.

  9. Digite seu nome de usuário e senha.

  10. Selecione Avançar.

Agora você deve encontrar seus contatos na agenda de endereços do seu iPhone.

Nota

A partir do iOS 12, é necessária uma criptografia SSL. Portanto, não desative o SSL (por esse motivo, um certificado é necessário em seu domínio, https://letsencrypt.org/ o fará).

Se ainda não estiver funcionando, dê uma olhada em Solução de problemas de contatos & Calendário ou Resolução de problemas de descoberta de serviço.
