匿名上传

您可以创建自己的个人目录，让其他人无需注册登录上传文件而无需开放访问修改权限。这是通过电子邮件、使用FTP服务器或使用商业文件共享服务发送大型附件的最佳选择。

设置自己的文件放置

转到“文件”并创建或选择文件夹，将将文件匿名上传到：

检查分享链接，允许编辑，隐藏文件列表

可以手动发送nextcloud上传文件夹链接或者使用nextcloud分享链接功能，此操作需要管理大哥许可。

上传文件中

匿名上传是如此简单，当你收到链接后仅需点击上传”Click to upload”按钮：

这将打开一个文件选择窗口让你选择上传文件，也支持拖动文件到窗口上传。
