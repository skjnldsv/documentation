IIndexOptions
=============


.. php:namespace:: OCP\FullTextSearch\Model

.. php:interface:: IIndexOptions


	.. rst-class:: phpdoc-description
	
		| Interface IIndexOptions
		
		| IndexOptions are created in FullTextSearch when an admin initiate an index
		| from the command line:
		| 
		| \./occ fulltextsearch:index "\{\\"option1\\": \\"value\\", \\"option2\\": true\}"
		
	
	:Source:
		`lib/public/FullTextSearch/Model/IIndexOptions.php#40 <https://github.com/nextcloud/server/blob/stable13/lib/public/FullTextSearch/Model/IIndexOptions.php#L40>`_
