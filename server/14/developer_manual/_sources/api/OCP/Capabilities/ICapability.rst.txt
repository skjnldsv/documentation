ICapability
===========


.. php:namespace:: OCP\Capabilities

.. php:interface:: ICapability


	.. rst-class:: phpdoc-description
	
		| Minimal interface that has to be implemented for a class to be considered
		| a capability\.
		
		| In an application use:
		|   $this\-\>getContainer\(\)\-\>registerCapability\(\'OCA\\MY\_APP\\Capabilities\'\);
		| To register capabilities\.
		| 
		| The class \'OCA\\MY\_APP\\Capabilities\' must then implement ICapability
		
	
	:Implemented by:
		:php:class:`OC\\OCS\\CoreCapabilities` 
	
	
	:Source:
		`lib/public/Capabilities/ICapability.php#37 <https://github.com/nextcloud/server/blob/stable13/lib/public/Capabilities/ICapability.php#L37>`_
