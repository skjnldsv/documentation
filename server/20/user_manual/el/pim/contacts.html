Χρήση της εφαρμογής Επαφές

Η εφαρμογή Επαφές δεν είναι ενεργοποιημένη από προεπιλογή στην Nextcloud | έκδοση | και πρέπει να εγκατασταθεί ξεχωριστά από το App Store.

Η εφαρμογή Nextcloud Contacts είναι παρόμοια με άλλες εφαρμογές επαφών για κινητά, αλλά με περισσότερη λειτουργικότητα. Ας δούμε βασικές δυνατότητες που θα σας βοηθήσουν να διατηρήσετε το βιβλίο διευθύνσεών σας στην εφαρμογή.

Παρακάτω, θα μάθετε πώς να προσθέτετε επαφές, να επεξεργάζεστε ή να καταργείτε επαφές, να ανεβάζετε μια εικόνα επαφής και να διαχειρίζεστε τα βιβλία διευθύνσεων της εφαρμογής επαφής.

Προσθήκη επαφών

Κατά την πρώτη πρόσβαση στην εφαρμογή Επαφές, ένα κενό προεπιλεγμένο βιβλίο διευθύνσεων καθίσταται διαθέσιμο.

Για να προσθέσετε επαφές στο βιβλίο διευθύνσεών σας, μπορείτε να χρησιμοποιήσετε μία από τις ακόλουθες μεθόδους:

  • Εισαγωγή επαφών χρησιμοποιώντας ένα αρχείο εικονικής επαφής (VCF / Vcard)

  • Μη αυτόματη προσθήκη επαφών

Ο πιο γρήγορος τρόπος για να προσθέσετε μια επαφή είναι να χρησιμοποιήσετε ένα αρχείο εικονικής επαφής (VCF / Vcard).

Εισαγωγή εικονικών επαφών

Για εισαγωγή επαφών χρησιμοποιώντας ένα αρχείο VCF / Vcard:

  1. Βρείτε «Ρυθμίσεις» στο κάτω μέρος της αριστερής πλευρικής γραμμής, δίπλα στο κουμπί με το γρανάζι.

  2. Κάντε κλικ στο κουμπί με το γρανάζι. Θα εμφανιστεί το κουμπί «Εισαγωγή» της εφαρμογής Επαφές.

Σημείωση

Η εφαρμογή Επαφές υποστηρίζει μόνο την εισαγωγή των vCards έκδοσης 3.0 και 4.0.

  1. Κάντε κλικ στο κουμπί «Εισαγωγή» και ανεβάστε το αρχείο VCF / Vcard.

Αφού ολοκληρωθεί η εισαγωγή, θα δείτε τη νέα σας επαφή στο βιβλίο διευθύνσεών σας.

Μη αυτόματη προσθήκη επαφών

Εάν δεν μπορείτε να εισαγάγετε εικονικές επαφές, η εφαρμογή Επαφές σάς επιτρέπει να ** προσθέσετε επαφές ** χειροκίνητα.

Για να δημιουργήσετε μια νέα επαφή:

  1. Κάντε κλικ στο κουμπί «+ Νέα επαφή».

    Ανοίγει μια κενή νέα διαμόρφωση επαφών στο πεδίο Προβολή εφαρμογής.

  1. Καθορίστε τα νέα στοιχεία επικοινωνίας. Οι αλλαγές που κάνατε εφαρμόζονται αμέσως.

Επεξεργασία ή κατάργηση στοιχείων επικοινωνίας

Η εφαρμογή Επαφές σάς δίνει τη δυνατότητα να επεξεργαστείτε ή να αφαιρέσετε τα στοιχεία επικοινωνίας.

Για να επεξεργαστείτε ή να καταργήσετε στοιχεία επικοινωνίας:

  1. Μεταβείτε στη συγκεκριμένη επαφή που θέλετε να τροποποιήσετε.

  2. Επιλέξτε τις πληροφορίες στο πεδίο που θέλετε να επεξεργαστείτε ή να καταργήσετε.

  3. Κάντε τις τροποποιήσεις σας ή κάντε κλικ στον κάδο απορριμμάτων.

Οι αλλαγές ή οι καταργήσεις που κάνατε σε οποιαδήποτε στοιχεία επικοινωνίας εφαρμόζονται αμέσως.

Προσθήκη και διαχείριση βιβλίων διευθύνσεων

Κάνοντας κλικ στο κουμπί «Ρυθμίσεις» (γρανάζι) στο κάτω μέρος της αριστερής πλευρικής γραμμής παρέχει πρόσβαση στις ρυθμίσεις της εφαρμογής Επαφή. Αυτό το πεδίο εμφανίζει όλα τα διαθέσιμα βιβλία διευθύνσεων, συγκεκριμένες επιλογές για κάθε βιβλίο διευθύνσεων και σας επιτρέπει να δημιουργήσετε νέα βιβλία διευθύνσεων, απλώς καθορίζοντας ένα όνομα βιβλίου διευθύνσεων.

Οι ρυθμίσεις επαφών είναι επίσης όπου μπορείτε να μοιραστείτε, να εξαγάγετε και να διαγράψετε βιβλία διευθύνσεων. Θα βρείτε τις διευθύνσεις URL CardDAV εκεί.

Δείτε: doc: index για περισσότερες λεπτομέρειες σχετικά με το συγχρονισμό των βιβλίων διευθύνσεών σας με iOS, macOS, Thunderbird και άλλους πελάτες του CardDAV.
