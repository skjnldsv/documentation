Sincronización con iOS

Calendario

  1. Abra a aplicación de axustes.

  2. Seleccione Calendario.

  3. Seleccione Contas.

  4. Seleccione Engadir conta.

  5. Seleccione Outro como tipo de conta.

  6. Seleccione Engadir conta CalDAV.

  7. Para o servidor, escriba o nome de dominio do servidor, é dicir example.com.

  8. Open Advanced Settings

  9. For server, type the domain name of your server and username i.e. example.com/remote.php/dav/principals/users/username/

  10. Close Advanced Settings

  11. Introduza o seu nome de usuario e contrasinal.

  12. Seleccione Seguinte.

O seu calendario agora será visíbel na aplicación Calendario.

Note

A partir do iOS 12 é necesario un cifrado SSL. Polo tanto, non desactive SSL (Por este motivo é necesario que o seu dominio dispoña dun certificado, pode obtelo en: https://letsencrypt.org/).

Contactos

  1. Abra a aplicación de axustes.

  2. Seleccione Contactos.

  3. Seleccione Contas.

  4. Seleccione Engadir conta.

  5. Seleccione Outro como tipo de conta.

  6. Seleccione Engadir conta CardDAV.

  7. Para o servidor, escriba o nome de dominio do servidor, é dicir example.com.

  8. Introduza o seu nome de usuario e contrasinal.

  9. Seleccione Seguinte.

Agora debe atopar os seus contactos no caderno de enderezos do seu iPhone.

Note

A partir do iOS 12 é necesario un cifrado SSL. Polo tanto, non desactive SSL (Por este motivo é necesario que o seu dominio dispoña dun certificado, pode obtelo en: https://letsencrypt.org/).

Se aínda non funciona, bótelle unha ollada a Solución de problemas de Contactos e Calendario ou a Solución de problemas de descubrimento do servizo.
