Nextcloud 웹 인터페이스

웹 브라우저로 Nextcloud 서버에 접속할 수 있습니다. URL을 통해 해당 Nextcloud 서버로 이동하고 (예: cloud.exmaple.com), 아이디와 암호를 입력하십시오:

웹 브라우저 요구사항

Nextcloud 웹 인터페이스의 원활한 동작과 우수한 사용자 경험을 위해, 아래 브라우저의 최신 버전 또는 Nextcloud가 지원하는 버전을 사용하십시오:

  • Google Chrome/Chromium (데스크탑 또는 안드로이드)

  • Mozilla Firefox (데스크탑 또는 안드로이드)

  • Apple Safari (데스크탑 또는 iOS)

  • Microsoft Edge

참고

해당 브라우저의 모든 버전을 지원하지는 않습니다. Nextcloud가 지원하는 버전에 대해서는 이 링크를 참조하십시오.

참고

영상 통화와 화면 공유 등 Nextcloud Talk의 원활한 사용을 위해서는 Mozilla Firefox 52 또는 Google Chrome/Chromium 49 이상을 사용하십시오.

경고

Microsoft Internet Explorer**는 **지원하지 않습니다.

사용자 인터페이스 탐색하기

기본적으로, Nextcloud 웹 인터페이스는 대시보드 혹은 파일 페이지를 표시합니다:

파일 페이지에서 파일을 추가, 제거, 공유할 수 있으며, 서버 관리자는 접근 권한을 변경할 수 있습니다.

Nextcloud 유저 인터페이스는 다음 기능으로 이루어져 있습니다:

  • 앱 선택 메뉴 (1): 좌상단에 위치해 있으며, 해당 Nextcloud에서 사용 가능한 모든 앱을 찾을 수 있습니다. 아이콘을 클릭하면 해당하는 앱으로 이동할 수 있습니다.

  • 앱 정보 영역 (2) : 좌측 사이드 바에 위치해 있으며, 선택된 앱과 관련된 필터 또는 작업을 표시합니다. 예를 들어 파일 앱을 사용하고 있다면, ‘공유된 파일’ 등 특정 파일의 빠른 탐색을 돕는 필터 집합이 표시됩니다. 다른 앱에서도 이와 유사한 항목들이 표시됩니다.

  • 애플리케이션 영역 (3): Nextcloud 유저 인터페이스의 중앙에 위치한 영역입니다. 선택한 앱의 컨텐츠 또는 사용자 기능 등이 표시됩니다.

  • 내비게이션 바 (4): 애플리케이션 영역 상단에 위치해 있으며, 최상위 위치로부터 현재 위치까지의 경로를 바 형태로 표시합니다.

  • 새로 만들기 버튼 (5): 내비게이션 바에 위치해 있으며, 새로운 파일, 폴더를 생성하고 파일을 업로드 할 수 있습니다.

참고

또한 Windows 탐색기, Mac Finder 등의 파일 관리자에서 Nextcloud의 파일 앱에 드래그 하여 특정 파일을 업로드할 수도 있습니다.

  • 검색 영역 (6): 우상단에 있는 돋보기 아이콘을 클릭하여 파일과 앱을 찾을 수 있습니다.

  • 연락처 메뉴 (7): 클라우드에 있는 사용자 및 연락처에 대한 개요를 제공합니다. 선택한 사용자 및 사용 가능한 앱에 따라, 곧바로 화상통화를 시작하거나 이메일을 전송할 수 있습니다.

  • 바둑판식 보기 버튼 (8): 작은 사각형 4개로 이루어진 아이콘이며, 폴더와 파일에 대한 바둑판식 보기를 설정/해제할 수 있습니다.

  • 설정 메뉴 (9): 검색 영역 우측에 있는 프로필 사진을 클릭하여 설정 드롭다운 메뉴를 엽니다. 설정 페이지는 다음의 기능을 제공합니다:

    • 데스크탑 또는 모바일 앱 다운로드 링크

    • 저장소 용량 사용량

    • 암호 관리

    • 이름, 이메일, 프로필 사진 설정

    • 연결된 인터넷 브라우저와 기기 관리

    • 그룹 회원

    • 인터페이스 언어 설정

    • 알림 관리

    • 연합 클라우드 아이디와 소셜 미디어 공유 버튼

    • 외부 저장소에 대한 SSL/TLS 증명서 관리

    • 2단계 인증 설정

    • Nextcloud 버전 정보

더 많은 정보는 여러분의 선호에 맞게 설정하는 방법 를 참조하십시오.
