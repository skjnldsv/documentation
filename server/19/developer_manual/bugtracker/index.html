Bugtracker

Thank you for helping Nextcloud by reporting bugs. Before submitting an issue, please read Issue submission guidelines first.

  • If the issue is with the Nextcloud server, report it to the Server repository

  • If the issue is with the Nextcloud desktop client, report it to the Desktop repository

  • If the issue is with the Nextcloud Android client, report it to the Android repository

  • If the issue is with the Nextcloud iOS client, report it to the iOS repository

  • If the issue with with a Nextcloud app, report it to where that app is developed

  • If the app is listed in our main GitHub organization report it to the correct sub repository
