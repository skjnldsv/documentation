Synchronisierung mit iOS

Kalender

  1. Öffnen Sie die Einstellungsanwendung.

  2. Kalender wählen.

  3. Konten wählen.

  4. Wählen Sie Konto hinzufügen.

  5. Wählen Sie Andere als Kontotyp.

  6. Wählen Sie CalDAV-Konto hinzufügen.

  7. Für Server geben Sie den Domain-Namen Ihres Servers ein, z.B. „example.com“.

  8. Open Advanced Settings

  9. For server, type the domain name of your server and username i.e. example.com/remote.php/dav/principals/users/username/

  10. Close Advanced Settings

  11. Geben Sie Ihren Benutzernamen und Ihr Passwort ein.

  12. Wählen Sie Weiter.

Ihr Kalender wird jetzt in der Kalenderanwendung angezeigt.

Bemerkung

Beginnend mit iOS 12 ist eine SSL-Verschlüsselung zwingend notwendig. Deaktivieren Sie daher nicht SSL (Aus diesem Grund ist ein Zertifikat auf Ihrer Domain erforderlich, https://letsencrypt.org/ genügt).

Kontakte

  1. Öffnen Sie die Einstellungsanwendung.

  2. Kontakte wählen.

  3. Konten wählen.

  4. Wählen Sie Konto hinzufügen.

  5. Wählen Sie Andere als Kontotyp.

  6. Wählen Sie CardDAV-Konto hinzufügen.

  7. Für Server geben Sie den Domain-Namen Ihres Servers ein, z.B. „example.com“.

  8. Geben Sie Ihren Benutzernamen und Ihr Passwort ein.

  9. Wählen Sie Weiter.

Sie sollten Ihre Kontakte jetzt im Adressbuch Ihres iPhones vorfinden.

Bemerkung

Beginnend mit iOS 12 ist eine SSL-Verschlüsselung zwingend notwendig. Deaktivieren Sie daher nicht SSL (Aus diesem Grund ist ein Zertifikat auf Ihrer Domain erforderlich, https://letsencrypt.org/ genügt).

Wenn es immer noch nicht funktioniert, lesen Sie Troubleshooting Contacts & Calendar oder _Troubleshooting Service Discovery.
