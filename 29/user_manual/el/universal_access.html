Καθολική Πρόσβαση

Η καθολική πρόσβαση είναι πολύ σημαντική σε εμάς. Ακολουθούμε πρότυπα ιστού και ελέγχουμε να είναι όλα λειτουργικά τόσο με το πληκτρολόγιο όσο και με βοηθητικό λογισμικό όπως οι αναγνώστες οθόνης. Σκοπεύουμε να συμμορφωνόμαστε με τις Οδηγίες προσβασιμότητας περιεχομένου Ιστού 2.1 σε επίπεδο ΑΑ, με το θέμα υψηλής αντίθεσης ακόμη και σε επίπεδο AAA.Ακολουθουμε επίσης τις γερμανικές BITV 2.0 οδηγίες.

Εάν βρείτε προβλήματα, μη διστάσετε να τα αναφέρετε στο πρόγραμμα παρακολούθηση προβλημάτων μας <https://github.com/nextcloud/server/issues/>`_.Και αν θέλετε να εμπλακείτε, ελάτε να γίνετε μέλος της ομάδας σχεδιασμού μας<https://nextcloud.com/design>`_!

Εστίαση και ταχύτητα ανταπόκρισης.

Το περιβάλλον της Nextcloud ανταποκρίνεται πλήρως και είναι αξιοποιήσιμο σε οποιοδήποτε μέγεθος οθόνης. Μπορείτε να μεγεθύνετε και να μικρύνετε και να ταιριάξετε το κείμενο και το μέγεθος των στοιχείων στις προτιμήσεις σας. Η πλοήγηση και η πλευρική μπάρα μπορούν να επεκταθούν ή να συμπτυχθούν.

Πλοήγηση μέσω πληκτρολογίου.

Μπορείτε να προηγηθείτε στo περιβάλλον Ιστού με το πληκτρολόγιο μόνο όπως μπορείτε με το ποντίκι:

  • Tab και ``Shift + Tab``για μετακίνηση μεταξύ των στοιχείων

  • Enter ή Space για να ενεργοποιήσετε ή να ανοίξετε το στοιχείο (ανάλογα με τον τύπο του στοιχείου)

  • ``Escape``που θα χρησιμοποιηθεί για το κλείσιμο των modals, των μενού popover και των προγραμμάτων προβολής αρχείων

  • Left arrow και``Right arrow`` για πλοήγηση μεταξύ φωτογραφιών στο πρόγραμμα προβολής

  • Ctrl + F για να εστιάσετε το πεδίο αναζήτησης

  • Ctrl + S για να αποθηκεύσετε αλλαγές σε προγράμματα επεξεργασίας όπως το Nextcloud Text

Για γρηγορότερη πλοήγηση, προσφέρουμε 2 «skip links» στην αρχή του εγγράφου τα οποία σας επιτρέπουν να:

  • Μετάβαση στο κύριο περιεχόμενο

  • Μετάβαση στην πλοήγηση της εφαρμογής

Το Nextcloud Talk έχει συντομεύσεις που τεκμηριώνονται επίσης μέσα στις ρυθμίσεις της ίδιας της εφαρμογής:

  • C για να εστιάσετε το πεδίο εισαγωγής μηνύματος

  • ``Escape``για να αποστιασετε το πεδίο εισαγωγής μηνύματος για να μπορείτε να χρησιμοποιήσετε συντομεύσεις

  • F για πλήρη οθόνη της συνομιλίας ή της κλήσης

  • Κατά τη διάρκεια μιας κλήσης:

    • M για να ενεργοποιήσετε και να απενεργοποιήσετε το μικρόφωνο

    • V για να ενεργοποιήσετε και να απενεργοποιήσετε το βίντεο

    • Space για push to talk ή push to mute

    • R για να σηκώσετε ή να κατεβάσετε το χέρι

Το Nextcloud Mail έχει συντομεύσεις, επίσης τεκμηριωμένες μέσα στις ρυθμίσεις της ίδιας της εφαρμογής:

  • C για τη σύνταξη νέου μηνύματος

  • Left arrow to switch to a newer message

  • Right arrow για εναλλαγή σε παλαιότερο μήνυμα

  • S για εναλλαγή ενός μηνύματος ως αγαπημένου

  • U για εναλλαγή ενός μη αναγνωσμένου μηνύματος

  • Del για να διαγράψετε ένα μήνυμα

  • Ctrl + Enter για αποστολή

  • R για ανανέωση και φόρτωση νέων μηνυμάτων

Περιλαμβανόμενα θέματα

Προσφέρουμε πολλά θέματα που μπορείτε να ενεργοποιήσετε για να βοηθήσετε την προσβασιμότητα:

  • Θέμα υψηλής αντίθεσης: Λειτουργία υψηλής αντίθεσης για διευκόλυνση της πλοήγησής σας. Η οπτική ποιότητα θα μειωθεί αλλά η διαύγεια θα αυξηθεί.

  • Σκούρο θέμα: Ένα σκούρο θέμα για να χαλαρώσετε τα μάτια σας μειώνοντας τη συνολική ακτινοβολία και φωτεινότητα. Είναι ακόμη υπό ανάπτυξη, επομένως αναφέρετε τυχόν προβλήματα που ενδέχεται να βρείτε.

  • Γραμματοσειρά Dyslexia: Το OpenDyslexic είναι μια δωρεάν γραμματοσειρά που έχει σχεδιαστεί για να μετριάσει ορισμένα από τα κοινά λάθη ανάγνωσης που προκαλούνται από τη δυσλεξία.

Για να φτάσετε στις ρυθμίσεις προσβασιμότητας:

  1. Ανοίξτε το μενού ρυθμίσεων στο τέλος της επικεφαλίδας

  2. Επιλέξτε Ρυθμίσεις

  3. Στην πλοήγηση, επιλέξτε Προσβασιμότητα

Σημείωση

Η αντίθεση των στοιχείων μπορεί να διαφέρει ανάλογα με το προσαρμοσμένο θέμα. Για παράδειγμα, το κύριο χρώμα θέματος χρησιμοποιείται ως χρώμα φόντου από την κεφαλίδα, τη σελίδα σύνδεσης και τα κύρια κουμπιά. Εάν αυτό προκαλεί προβλήματα αντίθεσης, επικοινωνήστε με τον διαχειριστή σας για βοήθεια.
